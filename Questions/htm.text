*** HTML Assignment
** Instructions
* Create an HTML webpage named assignment.html that demonstrates the following tags and their uses:

1. <html>, <head>, <title>, <body>

4. <hr> (horizontal line)
5. <b> / <strong> (bold text)
6. <i> / <em> (italic text)
7. <u> (underlined text)
