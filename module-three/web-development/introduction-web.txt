// HTML(HyperText Markup Language) is a markup language used to create the structure of a webpage
// The structure of a webpage is the layout of the webpage and the content of the webpage
// It is divided into three parts: the head, the body and the footer
// The head contains the metadata of the webpage
// The body contains the content of the webpage
// The footer contains the footer of the webpage

// Def: Tags
// Htmls are written in the form of tags which are enclosed in angle brackets < >
// Tags are case insensitive
// Tags are divided into two types: paired tags and self closing tags
// Tags are paired tags and self closing tags 
// Paired tags are tags that have an opening and closing tag e.g <p> </p>
// Self closing tags are tags that have only one tag <img />

// Creating a basic html file
// 1. Create a file with the extension .html
// 2. Write the basic structure of an html file
// 3. Save the file and open it in the browser

The basic structure of the HTML file include the following

1. Document Tags
    i. <html></html> : all tags must be inclosed into this tag this the root of the html tags
    ii. <head></head>: is used to hold the <title>, <style>, <meta>, <scripts> tags 
    iii . <body></body>: this contains the content that will be displayed on the browser

2. Heading and Text tags
    i. (<h1>,<h2>,<h3>,<h4>, <h5>, <h6>) heading tags: are used for heading tags for smaller or larger fonts 
    ii. <p> paragraph tag: is used to determine a paragraph in html
    iii <br> → Line break (self closing tag).
    iv. <hr> → Horizontal line (divider).
    v. <b> / <strong> → Bold text (strong has semantic importance).
    vi. <i> / <em> → Italic text (emphasis has semantic meaning).
    vii. <u> → Underlined text.
    viii. <span> → Inline container for styling or grouping.
3. Links & Navigation
    i. <a href="url"> anchor: → Creates a hyperlink.
        This used the attribute href for resource location
    ii. <nav> → Defines navigation links section.
4. Lists:
    To display on the browser we can use the ordered or unordered list display
    i. <ul> → Unordered list (bullets).
    ii. <ol> → Ordered list (numbers).
    iii. <li> → List item.
5. Images & Media
   i. <img src="file.jpg" alt="description"> → Displays an image.
   ii. <audio controls> → Embed audio.
   iii. <video controls> → Embed video.

6. Tables
    i. <table> → Defines a table.
    ii. <tr> → Table row.
    iii. <td> → Table data (cell).
    iv. <th> → Table header cell.
    v. <thead>, <tbody>, <tfoot> → Grouping table sections.

7. Forms & Input
    i. <form> → Form container.
    ii. <input type="text"> → Single-line input field.
    iii. <textarea> → Multi-line input.
    iv. <select> → Drop-down list.
    v. <option> → Item in a <select>.
    vi. <button> → Clickable button.
    vii. <label> → Label for input fields.

Attributes: tags can contain attributes which can either be used or not.
Example of an attribute is the href from the anchor tag.
